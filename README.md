# Backend-API

# Pre-reqs
- Install [.Net](https://dotnet.microsoft.com/download)

# Getting started
- Clone the repository
```bash
$ git clone https://github.com/JacielG/Backend-API/
```
- Run App on Dev Server
```bash
$ cd Backend-API
$ dotnet watch run
```
Process requests to `http://localhost:5000/`
